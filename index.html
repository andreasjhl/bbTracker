bbTracker

bbTracker is a simple application for Java-enabled smartphones to create GPS track logs. It tracks ones movement and provides that data for later analysis. It is not meant as a replacement for full-featured GPS software that does display maps and do path-finding, but rather a simple tool that does one thing and does it well.

This software is archived here and is no longer actively developed. Please note that this will not run on modern Android-based smartphones or BlackBerry phones with the BlackBerry 10 operating system. It's for Java ME enabled phones only (sometimes called "J2ME").

Installation

For most "normal" phones (feature-phones), pointing the phones browser to the JAD file will install the software.

For BlackBerry (OS 9 and earlier), pointing the browser at the JAR file will install the software.

The COD and ALX files are only provided for installation via the BlackBerry Desktop Manager and are usually not needed.

Download

The last released version os 0.5.3.1 and it's usually suggested to install this version: JAD (for classic phones), JAR (for BlackBerry).

Some users reported problems with the 0.5.3.1 release. If that version doesn't work for you, you can try the slightly buggier 0.5.3 release: JAD (for classic phones), JAR (for BlackBerry).

Installing any other versions is not recommended, but there's an archive of all released versions here.
